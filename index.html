HTML

  • HTML: hypertext markup language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • SRC contains location, or source, of the image file.
  • ALT contains a text string that describes the appearance and functionality of the image.
  • Header contains a navigation menu and/or introduction content.
  • Footer contains the author, copyright, contact, sitemap, and navigation.
  • HTML includes text elemnts, image elements, header elements, and list elements.
  • Elements used to divide and describe a page are; meta-related elements, content division elements, and semantic elements.
  • Style defines CSS styles associated with a website
  • Base defines the base of the URL for a webpage.
  • Link connects external resource to HTML document
  • Meta defines metadata; character det, description, keywords, author and viewpoint.
  • Script embeds client side scripts in the HTML document.
