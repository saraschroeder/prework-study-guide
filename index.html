Prework Study Guide

HTML

  • HTML: hypertext markup language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • SRC contains location, or source, of the image file.
  • ALT contains a text string that describes the appearance and functionality of the image.
  • Header contains a navigation menu and/or introduction content.
  • Footer contains the author, copyright, contact, sitemap, and navigation.
  • HTML includes text elemnts, image elements, header elements, and list elements.
  • Elements used to divide and describe a page are; meta-related elements, content division elements, and semantic elements.
  • Style defines CSS styles associated with a website
  • Base defines the base of the URL for a webpage.
  • Link connects external resource to HTML document
  • Meta defines metadata; character det, description, keywords, author and viewpoint.
  • Script embeds client side scripts in the HTML document.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • 3 ways to style a webpage using CSS: Inline CSS, Internal CSS style sheet, External style sheet.
  • When styling an element directly in the HTML file, inline CSS is used.
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • An external CSS style sheet is the most common way to style a webpage.
  • Declarations contain two important components; the CSS property we want to apply and the value of the property.

Git

  • Git Status: Checks what branch we are currently on.
  • Git Checkout -b branch-name: Creates a new branch and switches to it.
  • Git push origin feature/branch-name: brings changes to remote branch.
  • Git pull: pulls any changes
  • -A adds all changes.
